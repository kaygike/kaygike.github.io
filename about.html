Tech Stack

Webpages are edited in plain text using Markdown syntax.

A small Bash script is run to recursively convert the Markdown files to HTML using Pandoc. A custom Pandoc Template is used to define the HTML structure, and all accompanying CSS and images were made from scratch.

All files are committed to a Git repository, and the HTML files generated via Pandoc are served directly from there.
